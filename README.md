# FicelloEngine
A small open source C++ SDL2 game engine for linux with lua as scripting langage

## Downloading dependencies
Arch Linux:

```bash
pacman -S sdl2
pacman -S base-devel
pacman -S cmake
```

Debian:

```bash
apt-get install build-essentials
apt-get install libsdl2-dev libsdl2-2.0-0
```

## Building

```bash
cd FicelloEngine/build
chmod +x build.sh
./build.sh
```
